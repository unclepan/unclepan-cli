
# unclepan-cli
初始化模版的工具

# 安装
```
npm install unclepan-cli -g
```

# 使用
打开控制台输入`unclepan`或者 `unclepan -h`，你会看到以下信息
```
  Usage: unclepan <command> [options]

  Options:

    -V, --version  output the version number
    -h, --help     output usage information

  Commands:

    add                  添加一个新的模板！
    list                 列出所有可用模板！
    init                 初始化一个新的项目！
    delete               删除一个模板！

```
